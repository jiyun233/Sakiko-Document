# 成绩工具合集

这里包含所有常用的成绩与计算类功能。

---

## ⭐ B50 生成

```text
b50 [qq]
```

- 不填 QQ → 生成自己的 B50
- 填 QQ → 生成目标用户的 B50
- 支持at群友查询

## 🧮 Rating / 定数

### 定数查歌
```
定数查歌 <等级/定数>
```

---

### Rating 计算
```
rating <定数> [分数]
```

---

### OverPower
```
overpower <定数> <分数> [fc]
```

## 随机歌曲

```text
随机 <等级/定数>
```

随机抽取一首对应难度的曲子。

## 曲名查询

模糊查歌：

```
<关键词>是什么歌
```

## 谱面预览

```
谱面预览 <难度> <id/别名>
```

未指定难度默认查询MASTER
暂不支持World's End
例如:

```
谱面预览 黑AIR
谱面预览 奶龙红
谱面预览 大鸟居
```

## 🔧 tolerant 容错计算

```
tolerant <分数> <物量/歌曲>
```

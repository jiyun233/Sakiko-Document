# 绑定账号

使用 本Bot 前，你需要先绑定 **落雪查分器 Token** 并且 **游玩并同步** 过中二节奏成绩

---

## 📌 获取 Token

1. 前往落雪查分器：  
   https://maimai.lxns.net
2. 注册并登录
3. 前往 **账号详情** -> **第三方应用** -> **个人 API 密钥** 复制Token
4. 试用Bot命令进行账号绑定

---

## 🔧 绑定命令

```text
bind <token>
```

示例：

```text
bind 123456789abcdef
```

成功后 Bot 将自动关联你的查询数据。

---

## ❗ 注意事项

- Token 请勿泄露
- 若 Token 失效，可随时重新绑定
- 若绑定报错，可向群管理或开发者反馈

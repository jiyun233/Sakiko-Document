# Sakiko-ChuniBot 文档

一个专为玩家设计的多功能舞萌 / 中二查分 Bot。  
提供便捷的查分器切换、定数工具、牌子进度、机厅人数等功能。

> 本文档适用于 **Sakiko-ChuniBot v1.4**

## 🚀 快速开始

要使用大部分功能，请先完成：

1. 前往 **落雪查分器** 注册账号
2. 获取 Token 并在 Bot 中绑定
3. 即可使用全部功能

## 📚 文档结构

- 使用指南（Guide）
    - 绑定账号
    - 切换查分器
    - 机厅人数
    - 牌子进度
    - 成绩工具
    - 其它功能

# 牌子进度查询

可查询不同版本的牌子达成情况。

## 🔧 使用方法

```text
progress <版本> [牌子]
```

示例：

```text
progress luminous 将
```

---

## 📌 默认牌子

若未指定，默认查询将牌进度

支持查询进度: 

- SSS (将/tribute)
- S (spirit)
- All Justice (神/legend)

支持版本/牌子试用别名查询
例如:

```text
prog lmnp 神
prog vrs 神
prog amz+ S
```

## ⚠ 注意事项

- 必须 **先绑定落雪 Token** 才能使用
- 若查无数据，请检查是否：
    - 版本名正确
    - 落雪数据是否同步
